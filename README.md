# Justin Rhoy Platon — Portfolio

Brief personal portfolio site showcasing projects, skills, and services. This is a lightweight, responsive static website built with plain HTML, CSS and JavaScript.

## Tech stack
- HTML5
- CSS3 (Flexbox & Grid)
- Vanilla JavaScript
- SVG icons and responsive images

## Main files
- `index.html` — site entry and content
- `css/styles.css` — all styles, responsive rules, and UI components
- `js/main.js` — interactive behavior (nav toggle, smooth scroll, small helpers)
- `assets/` — images, icons, and `resume.pdf`

## Usage
- Open `index.html` in a browser to view the site locally.
